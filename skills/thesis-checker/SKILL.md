# Thesis Checker

## Overview

本技能用于自动审查毕业论文的格式合规性和内容质量，根据学校提供的论文格式要求和毕业论文规范，对学生提交的论文进行多维度检查，精准定位问题位置并给出可操作的修改建议，帮助学生快速完成论文格式调整和内容优化。

## 核心功能

本技能支持以下检查维度：

### 1. 格式合规性检查
- **页面格式**：页边距、行距、页码、页眉页脚、纸张大小
- **字体格式**：标题层级字体、正文字体、字号、字间距、加粗/斜体规范
- **段落格式**：首行缩进、对齐方式、段前段后间距
- **引用格式**：参考文献格式、脚注/尾注格式、引用标注规范
- **图表格式**：图片/表格编号、标题位置、图例规范、表格边框要求

### 2. 内容质量检查
- **结构完整性**：摘要、关键词、目录、正文、参考文献、附录等部分是否齐全
- **逻辑连贯性**：章节结构是否合理、论点是否清晰、论证是否充分
- **格式一致性**：同一级标题格式是否统一、术语使用是否一致
- **引用规范性**：参考文献是否齐全、引用标注是否对应

### 3. 特殊要求检查
- 学校特定的格式要求（如封面、声明页、致谢页格式）
- 专业特定的内容要求（如字数限制、研究方法要求）
- 重复率检测提示（如果提供查重报告）

## 使用流程

### 第一步：收集规范文件
1. 要求用户提供学校官方的《毕业论文格式要求》文档
2. 要求用户提供所在专业的《毕业论文写作规范》
3. 将以上文件保存到 `references/` 目录下，命名为：
   - `references/school-format-requirements.md`
   - `references/major-thesis-requirements.md`

### 第二步：接收论文文件
支持的论文格式：
- .docx 格式（优先推荐，可精确定位行号和位置）
- .pdf 格式（需要OCR识别，位置定位可能存在偏差）
- .md 格式（纯文本格式，支持格式检查）

### 第三步：执行检查流程
1. **格式检查**：逐页对照格式要求，检查所有格式规范
2. **内容检查**：通读论文，检查结构完整性和逻辑连贯性
3. **问题定位**：对每个问题标注具体位置（章节、页码、行号）
4. **建议生成**：针对每个问题给出具体的修改建议和参考示例

### 第四步：输出检查报告
检查报告包含以下内容：
- 总体评分和合规性评估
- 问题分类统计（格式问题、内容问题、引用问题等）
- 详细问题列表：位置 + 问题描述 + 修改建议
- 优先级建议：优先修改的严重问题和次要问题

## 资源说明

### scripts/
`check_thesis.py` - 论文自动检查脚本，支持docx、pdf、md格式论文的自动格式检查
- 依赖安装：`pip install python-docx PyPDF2 markdown`
- 使用方法：`python scripts/check_thesis.py 论文文件.docx -o 检查报告.md`

### references/
存放学校和专业的论文规范文件：
- `school-format-requirements.md` - 学校毕业论文格式要求（需用户提供）
- `major-thesis-requirements.md` - 专业毕业论文写作规范（需用户提供）
- `README.md` - 规范文件说明
