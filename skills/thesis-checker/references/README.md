# 论文规范参考目录

请将学校提供的论文规范文件放在此目录：

## 需要提供的文件：

1. `school-format-requirements.md` - 学校官方发布的《毕业论文格式要求》
   - 包含：页边距、字体、字号、行距、页码、页眉页脚等格式要求
   - 包含：标题层级、正文、引用、参考文献等具体规范
   - 包含：封面、摘要、目录、附录等部分的格式要求

2. `major-thesis-requirements.md` - 所在专业的《毕业论文写作规范》
   - 包含：论文结构要求、字数要求、研究方法要求
   - 包含：专业特定的内容要求、引用规范
   - 包含：查重要求、答辩相关要求

## 文件格式要求：
- 优先使用Markdown格式（.md），便于AI识别
- 如果是Word或PDF格式，请转换为文本格式后保存
- 重要的格式示例可以截图保存到 assets/ 目录下
